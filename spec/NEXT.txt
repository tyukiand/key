10. Implement the unredacted-allowlist (spec §B.9): (done)

    - sanitizing constructor (no empty literals; no
      whitespace-only); no regex matchers (literal-only);
    - `redact_value` returns the value verbatim iff the
      value matches a Value entry exactly OR starts with
      a Prefix entry.

11. Wire `redact_value` into EVERY OsEffects read path (done)
    (spec §B.7):
    - env_vars() — per-value with name_hint;
    - read_file(), read_to_string() — line-by-line, no
      name_hint;
    - any other byte-returning read.

    Redaction uses the literal pattern "REDACTED42"
    looped, truncated to the ORIGINAL value length.
    Alphabet ⊂ base64 ∩ base64url. Reason for redaction
    goes to debug log only; never in the value.
